My MK8DX Track Times

This application helps track and visualise Mario Kart 8 Deluxe (MK8DX) personal bests (PBs) in Time Trials (TTs).

About

This is a personal project to improve my coding skills and visualise my MK8DX times. This web app has a variety of features to help visualise your current PBs and improvements.

View your timesheet to see all the tracks in a table format, plus some overall statistics and graphs. Use the update form to add new PBs or remove times. If you can't decide which track to play next, use the Track Picker to randomly choose one based on some criteria.

Note that many features will be incomplete, jank, or simply not implemented yet as I am learning :)
