My MK8DX Track Times

This application helps track and visualise Mario Kart 8 Deluxe personal bests in Time Trials.

About

This is a personal project to improve my coding skills and visualise my MK8DX TT improvements. This web app has a variety of features to help visualise your current PBs and different metrics.

View your timesheet to see all the tracks in a table format, plus some overall statistics and graphs. Each track has a link to an individual page with convenient features and an improvement graph. Use the update form to add new PBs or remove times. If you can't decide which track to play next, use the track picker to randomly choose one based on some criteria.

Note that some features will be incomplete, jank, or simply not implemented yet as this project is mainly for fun.

Glossary

The following terminology is well-known in the community but listed here for clarity.
  • MK8D / MK8DX - Mario Kart 8 Deluxe
  • TT / TA - time trial; time attack
  • Track code - an abbreviation used to identify a track; e.g., Mario Kart Stadium = MKS
  • PB / PR - personal best; personal record
  • WR - world record
  • BKT - best known time, usually in reference to a NITA WR
  • Shroom - TT using mushrooms
  • NITA - no item time attack, mushrooms are disallowed
  • Meta Only - only a specific kart combination is allowed; motion controls are disallowed
